사용자가 당일 달성해야 하는 미션을 등록하고 미션 완료 버튼을 터치할 때마다, 상단에 위치한 트랙 위의 자동차 아이콘이 실제 레이싱 하듯이 실시간으로 이동하며 성취감을 제공합니다.
기술적으로 수업 시간에서 실습한  StatefulWidget과 setState 상태 관리를 활용하여 (isMission = !isMission)로 미션 달성에 따른 변화를 실시간으로 화면에 반영해주는 코드를 작성하였습니다. 
또한 추가적으로  AnimatedAlign 위젯을 결합하여 자동차가 매끄럽게 수평 이동하도록 하였습니다. 기존의 Todo 리스트 앱에서 레이싱 게임 효과를 결힙하여 사용자가 목표 달성에 더욱 흥미를 가지고, 지속적으로 루틴을 실천할 수 있도록 하는 효과를 가진 어플리케이션 입니다.
